# Recordings list visible with concise name format

Ensure recordings list is visible in the Recordings tab and uses the concise name format with date/time, duration, and optional title.

## Acceptance Criteria

- Recordings list is visible in the Recordings tab by default.
- Recording display name format: "MMM d at h:mma - Xmin" or "MMM d at h:mma - Xmin - Title" (e.g., "Feb 7 at 5:05pm - 3min", "Feb 6 at 3:10pm - 31min - 1:1 with Alex").
- Optional title appears only when present.
